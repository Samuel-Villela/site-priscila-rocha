# Site Profissional - Priscila Rocha Psicóloga

Site profissional desenvolvido para divulgação dos serviços de psicologia da Priscila Rocha.

## Sobre o projeto

O objetivo deste site é apresentar informações profissionais, áreas de atuação, diferenciais, formas de contato e facilitar o agendamento de consultas.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages (hospedagem)

## Acesso

O site está disponível em:

https://samuel-villela.github.io/site-priscila-rocha/

## Funcionalidades

- Design responsivo
- Navegação intuitiva
- Integração com WhatsApp
- Mapa de localização
- SEO básico
- Sitemap XML
